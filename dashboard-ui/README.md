# CarbonTrack

CarbonTrack is a brutalist-themed, comprehensive carbon footprint tracking dashboard built with React, Vite, and TypeScript.

## Features
- **Dashboard**: Track your footprint trends, analyze your activity breakdowns, and receive personalized AI insights generated by Google's Gemini API based on your telemetry.
- **WebGL Visualization**: Experience a highly performant 3D WebGL globe representation of carbon hotspots, running smoothly alongside React.
- **Log Activities**: An intuitive interface to log daily actions across Transportation, Food, Energy, and Shopping, with dynamic CO2 estimation.
- **History & Reports**: A sortable, filterable history view, with interactive charts mapping emissions against goals.
- **Responsive & Accessible**: Mobile-first design, robust keyboard navigation, and full ARIA support ensuring a seamless experience for all users.

## Technology Stack
- **Frontend**: React 18, Vite, TypeScript
- **Styling**: Tailwind CSS v4, Framer Motion
- **Visualization**: Recharts, Three.js
- **Data Persistence**: LocalStorage API
- **AI Integration**: Google Gemini API via serverless backend function

## Running the Application

### Prerequisites
- Node.js (v18+)
- npm or yarn

### Setup
1. Clone the repository.
2. Run `npm install` in the `dashboard-ui` directory.
3. Configure your API key by copying `.env.example` to `.env` if necessary (in local dev, you can use `.env.local` to store `GEMINI_API_KEY`).
4. Run `npm run dev` to start the local server.

## Security & Architecture
- API keys are protected and never exposed to the client. The frontend communicates with a secure `/api/gemini` backend endpoint.
- Content is safely rendered using standard React mechanisms without exposing XSS vectors via `dangerouslySetInnerHTML`.
